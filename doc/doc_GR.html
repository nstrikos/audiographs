Πέτρου Α 4:10

Audiographs - Οδηγός χρήσης

Σημαντική σημείωση!!!

Πριν χρησιμοποιήσετε το audiographs, φροντίστε να ρυθμίσετε την ένταση ήχου σε χαμηλή τιμή,
επειδή η εφαρμογή μπορεί να παράγει πολύ υψηλές συχνότητες ήχου

Περιεχόμενα

  1. Γρήγορη εισαγωγή
  2. Οδηγός χρήσης
  3. Συντομεύσεις
  4. Υποστηριζόμενες μαθηματικές συναρτήσεις

1. Γρήγορη εισαγωγή

1. Ξεκινήστε την εφαρμογή
2. Πληκτρολογήστε x
3. Πατήστε Enter

Μόλις ακούσατε την ηχητική αναπαράσταση της συνάρτησης f(x) = x

Μερικές χρήσιμες σημειώσεις:

1. Η εφαρμογή διαθέτει αυτόνομο συνθέτη ομιλίας, γι' αυτό μπορείτε να απενεργοποιήσετε το πρόγραμμα ανάγνωσης οθόνης.
Για NVDA πατήστε το κουμπί NVDA + s.
Εάν δεν θέλετε να απενεργοποιήσετε το πρόγραμμα ανάγνωσης οθόνης, πατήστε F2 για να απενεργοποιήστε τη σύνθεση ομιλίας του audiographs

2. Κάθε φορά που θέλετε να μελετήσετε μια νέα συνάρτηση πατήστε Control + N για να ξεκινήσετε από την αρχή.

3. Όταν ξεκινάτε με μια νέα λειτουργία, τo audiographs θα ορίσει ένα όριο από -10 έως 10 και για τους άξονες x και y. Ήχος πάνω ή κάτω από αυτά τα όρια θα περικοπεί.

2. Οδηγός χρήσης

Εδώ θα βρείτε ασκήσεις αυξανόμενης δυσκολίας που θα σας βοηθήσουν να μάθετε τις λεπτομέρειες της εφαρμογής

Άσκηση 1. Μελετήστε τη συνάρτηση f(x) = 2*x

0. Πατήστε Control + N για να μελετήσετε μια νέα συνάρτηση. Αυτό δεν χρειάζεται εάν μόλις ξεκινήσατε την εφαρμογή.
1. Πατήστε 2 x
2. Πατήστε Enter
3. Ακούστε τον ήχο. Η συχνότητα ήχου ξεκινά από χαμηλά και ανεβαίνει ψηλά
4. Πατήστε F3 για να αλλάξετε από συνεχή ήχο σε μουσικές νότες
5. Πατήστε ξανά Enter. Θα ακούσετε μουσικές νότες, όχι ένα συνεχή ήχος
6. Πατήστε F4 για να χρησιμοποιήσετε εναλλακτικές μουσικές νότες για αρνητικές τιμές.
7. Πατήστε ξανά Enter. Θα ακούσετε ότι ο ήχος αλλάζει πέρα από ένα ορισμένο σημείο. Χρησιμοποιώντας αυτόν τον τρόπο, γνωρίζετε εάν είστε σε αρνητικές ή θετικές τιμές. Σημειώστε ότι αυτή η δυνατότητα μπορεί να χρησιμοποιηθεί μόνο όταν οι μουσικές νότες επιλέγονται με το F3
8. Πατώντας Page Up και Page Down, μπορείτε να εξερευνήσετε τη συνάρτηση. Πατήστε Page Up για να μεταβείτε στα δεξιά. Πατήστε Page Down για να μεταβείτε στα αριστερά
9. Πατώντας τα κουμπιά Home και End, μπορείτε να μεταβείτε στην αρχή και στο τέλος της συνάρτησης
10. Πατώντας Control + x, μπορείτε να ακούσετε τη συντεταγμένη x του τρέχοντος σημείου
11. Πατώντας Control + y, μπορείτε να ακούσετε τη συντεταγμένη y του τρέχονυος σημείου
12. Πατώντας Control + ], μπορείτε να αυξήσετε το βήμα της εξερεύνησης, κάνοντας έτσι την εξερεύνηση ταχύτερη
13. Πατώντας Control + [, μπορείτε να μειώσετε το βήμα της εξερεύνησης, κάνοντας έτσι την εξερεύνηση πιο αργή
14. Πατώντας Tab μπορείτε να εξερευνήσετε όλες τις ρυθμίσεις της εφαρμογής
15. Πατήστε Tab για να φτάσετε στο ελάχιστο x. Αυτή είναι η αρχή της συνάρτησης. Από προεπιλογή έχει οριστεί σε -10 όταν ο χρήστης επεξεργάζεται μια νέα συνάρτηση
16. Πατήστε Tab για να φτάσετε στο μέγιστο x. Αυτό είναι το τελικό σημείο της συνάρτησης. Από προεπιλογή ορίζεται στο 10 όταν ο χρήστης επεξεργάζεται μια νέα συνάρτηση
17. Πατήστε Tab για να φτάσετε στο ελάχιστο y. Αυτή είναι η χαμηλότερη τιμή που μπορεί εμφανιστεί στο γράφημα. Θα κοπούν τιμές μικρότερες από το ελάχιστο. Από προεπιλογή ορίζεται σε -10 όταν ο χρήστης επεξεργάζεται μια νέα συνάρτηση
18. Πατήστε Tab για να φτάσετε στο μέγιστο y. Αυτό είναι η υψηλότερη τιμή που μπορεί να εμφανιστεί στο γράφημα. Θα κοπούν τιμές μεγαλύτερες από το μέγιστο. Από προεπιλογή ορίζεται στο 10 όταν ο χρήστης επεξεργάζεται μια νέα συνάρτηση
19. Πατήστε Tab μέχρι να φτάσετε στη διάρκεια που από προεπιλογή είναι 15 δευτερόλεπτα. Τώρα μπορείτε να αυξήσετε τη διάρκεια του ήχου ή να τη μειώσετε διάρκεια
20. Πατήστε Tab για να φτάσετε στην ελάχιστη συχνότητα. Η ελάχιστη συχνότητα είναι η χαμηλότερη συχνότητα που θα παραχθεί από την εφαρμογή. Προσαρμόστε στην τιμή που προτιμάτε.
21. Πατήστε Tab για να φτάσετε στη μέγιστη συχνότητα. Η μέγιστη συχνότητα είναι η υψηλότερη συχνότητα που θα παραχθεί από την εφαρμογή. Προσαρμόστε στην τιμή που προτιμάτε.
22. Οι τιμές ελάχιστης και μέγιστης συχνότητας χρησιμοποιούνται μόνο όταν οι μουσικές νότες δεν επιλέγονται. Διαφορετικά η εφαρμογή θα παρακάμψει αυτές τις ρυθμίσεις με συχνότητες μουσικών νότων.
23. Πατώντας F9 μπορείτε να μειώσετε τα δεκαδικά ψηφία ακριβείας
24. Πατώντας F10 μπορείτε να αυξήσετε τα δεκαδικά ψηφία ακριβείας
25. Επιλέξτε Επαναφορά ρυθμίσεων ήχου για να επαναφέρετε τις προεπιλεγμένες ρυθμίσεις ήχου
26. Ανά πάσα στιγμή, πατήστε Control + E, για να επιστρέψετε στην επεξεργασία της συνάρτησης

Άσκηση 2. Μελετήστε τη συνάρτηση f(x) = x^2 + 1

0. Εκκινήστε την εφαρμογή ή πατήστε Control συν N για να ξεκινήσετε τη μελέτη μιας νέας συνάρτησης εάν έχετε ήδη μελετήσει μια συνάρτηση
1. Πατήστε x δύναμη 2 συν 1
2. Πατήστε Enter
3. Ακούστε τον ήχο, η συνάρτηση ξεκινά από ψηλά, χαμηλώνει και μετά πηγαίνει πάλι ψηλά
4. Πατήστε Home για να μεταβείτε στο σημείο εκκίνησης.
5. Πατήστε Control + x για να ακούσετε τh συντεταγμένη x. Θα ακούσετε -10
6. Πατήστε Control + y για να ακούσετε τη συντεταγμένη y. Θα ακούσετε το 101. Άρα η συνάρτηση ξεκινά από το σημείο (-10, 101)
7. Πατήστε F3 για εναλλαγή από συνεχή ήχο σε ηχητικές σημειώσεις.
8. Πατήστε Control + δεξί βέλος για να μεταβείτε στο επόμενο σημείο μελέτης.
9. Η συχνότητα του ήχου θα αρχίσει να μειώνεται μέχρι να ακούσετε "τοπικό ελάχιστο"
10. Πατήστε Control + x, θα ακούσετε 0,00
11. Πατήστε Control + y, θα ακούσετε 1.00, άρα υπάρχει ένα τοπικό ελάχιστο στο (0, 1)
12. Πατήστε Control + δεξί βέλος
13. Η συχνότητα του ήχου θα αρχίσει να αυξάνεται μέχρι να ακούσετε τη φράση "ending σημείο"
14. Πατήστε control + x, θα ακούσετε 10
15. Πατήστε control + y, θα ακούσετε 101
16. Άρα η συνάρτηση ξεκινά από (-10, 101) κατεβαίνει στο (0, 1) και στη συνέχεια μέχρι (10, 101)
17. Από τις τιμές που θα ακούσετε από την εφαρμογή εξαρτώνται τα ψηφία ακριβείας που έχετε επιλέξει. Πατήστε F9 για μείωση και F10 για αύξηση ψηφίων ακριβείας

Άσκηση 3. Μελετήστε τη συνάρτηση f(x) = -x^2 - 1

0. Ξεκινήστε την εφαρμογή ή πατήστε Control συν N για να ξεκινήσετε μια νέα συνάρτηση, εάν έχετε ήδη μελετήσει μια συνάρτηση

2. Πατήστε F3 για να επιλέξετε συνεχή ήχο ή μουσικές νότες ανάλογα με τις προτιμήσεις σας
3. Πατήστε Enter. Ακούστε τον ήχο, η λειτουργία ξεκινά από χαμηλά, πηγαίνει ψηλά και μετά πάλι χαμηλά
4. Πατήστε Home για να μεταβείτε στο σημείο εκκίνησης.
5. Πατήστε Control + x για να ακούσετε τη συντεταγμένη x. Θα ακούσετε -10
6. Πατήστε Control + y για να ακούσετε τη συντεταγμένη y. Θα ακούσετε -101. Άρα η συνάρτηση ξεκινά στο σημείο (-10, -101)
7. Πατήστε Control + δεξί βέλος για να μεταβείτε στο επόμενο σημείο μελέτης.
8. Η συχνότητα του ήχου θα αρχίσει να αυξάνεται μέχρι να ακούσετε "τοπικό μέγιστο"
9. Πατήστε Control + x, θα ακούσετε 0,00
10. Πατήστε Control + y, θα ακούσετε -1,00, οπότε υπάρχει ένα τοπικό μέγιστο στο (0, -1)
11. Πατήστε Control + δεξί βέλος
12. Η συχνότητα του ήχου θα αρχίσει να μειώνεται μέχρι να ακούσετε "τελικό σημείο"
13. Πατήστε control + x, θα ακούσετε 10
14. Πατήστε control + y, θα ακούσετε -101
15. Άρα η συνάρτηση ξεκινά από το (-10, -101) ανεβαίνει στο (0, -1) και στη συνέχεια προς τα κάτω στο (10, -101)
16. Οι τιμές που θα ακούσετε από την εφαρμογή εξαρτώνται από τα ψηφία ακριβείας που έχετε επιλέξει. Πατήστε F9 για μείωση και F10 για αύξηση των ψηφίων ακριβείας

Άσκηση 4. Μελετήστε τη συνάρτηση f(x) = x^2 - 8x + 8

0. Ξεκινήστε την εφαρμογή ή πατήστε Control συν N για να ξεκινήσετε μια νέα συνάρτηση εάν έχετε ήδη μελετήσει μια συνάρτηση
1. Πατήστε x δύναμη 2 μείον 8 x συν 8
2. Πατήστε Enter. Ακούστε τον ήχο, η λειτουργία ξεκινά από ψηλά, έπειτα πηγαίνει χαμηλά και μετά πάλι ψηλά
3. Πατήστε F3 και βεβαιωθείτε ότι έχετε επιλέξει "χρήση μουσικών νότων"
4. Πατήστε F4 και βεβαιωθείτε ότι έχετε επιλέξει "χρήση εναλλακτικών νότων για αρνητικές τιμές". Αυτή η ρύθμιση μπορεί να χρησιμοποιηθεί μόνο όταν έχετε επιλέξει "χρήση μουσικών νότων". Δεν μπορεί να χρησιμοποιηθεί με συνεχή ήχο. Τώρα θα ακούσουμε διαφορετικούς ήχους όταν η συνάρτηση έχει αρνητικές τιμές.
5. Πατήστε Enter. Θα ακούσετε τη συνάρτηση να ξεκινά από ψηλά, έπειτα να πηγαίνει χαμηλά. Κάποια στιγμή ο ήχος θα αλλάξει. Αυτό σημαίνει ότι η συνάρτηση έχει αρνητικές τιμές. Μετά ανεβαίνει ξανά και κάποια στιγμή αλλάζει πάλι σε κανονικό ήχο. Αυτό σημαίνει ότι η συνάρτηση έχει θετικές τιμές και συνεχίζει να ανεβαίνει
6. Πατήστε Home. Θα ακούσετε ότι βρισκόμαστε στο σημείο εκκίνησης.
7. Πατήστε Ctrl + x, θα ακούσετε -10,00
8. Πατήστε Ctrl + y, θα ακούσετε 188
9. Πατήστε F5 και βεβαιωθείτε ότι έχετε επιλέξει το "Πάυση ήχου στο μηδέν".
10. Πατήστε Control + δεξί βέλος για να μεταβείτε στο επόμενο σημείο μελέτης. Θα ακούσετε "μηδέν".
11. Πατήστε Ctrl + x, θα ακούσετε 1.17
12. Πατήστε Ctrl + y, θα ακούσετε 0,00. Άρα η συνάρτηση έχει ρίζα στο (1.17, 0).
13. Πατήστε Control + δεξί βέλος για να μεταβείτε στο επόμενο σημείο μελέτης. Θα ακούσετε "τοπικό ελάχιστο". Θα ακούσετε διαφορετικό ήχο γιατί είμαστε κάτω από το μηδέν.
14. Πατήστε Control + x, θα ακούσετε 4
15. Πατήστε Control + y, θα ακούσετε -8. Άρα η συνάρτηση έχει τοπικό ελάχιστο στο (4, -8)
16. Πατήστε Control + δεξί βέλος για να μεταβείτε στο επόμενο σημείο μελέτης. Θα ακούσετε "μηδέν".
17. Πατήστε Ctrl + x, θα ακούσετε 6,83
18. Πατήστε Ctrl + y, θα ακούσετε 0,00. Άρα η συνάρτηση έχει ρίζα στο (6.83, 0).
19. Πατήστε Control + δεξιά, θα ακούσετε "τελικό σημείο"
20. Πατήστε Control + x, θα ακούσετε 10
21. Πατήστε Control + y, θα ακούσετε 27.98
22. Άρα η συνάρτηση ξεκινά από το (-10, 188) κατεβαίνει στο (4, -8) και μετά ανεβαίνει μέχρι το (10, 27.98)
23. Πατήστε Home. Είμαστε πάλι στην αρχή.
24. Πατήστε Control + 1. Τώρα μπορούμε να ακούσουμε την πρώτη παράγωγο της συνάρτησης.
25. Πατήστε Control + δεξιά. Ακούτε αρνητικές τιμές μέχρι να φτάσετε στο τοπικό ελάχιστο. Αυτό σημαίνει ότι η συνάρτηση μειώνεται
26. Πατήστε Control + δεξιά. Ακούτε θετικές τιμές μέχρι να φτάσετε στο τελικό σημείο. Αυτό σημαίνει ότι η συνάρτηση αυξάνεται
27. Πατήστε Control + 0 εάν θέλετε να αφήσετε την λειτουργία πρώτης παραγώγου και να επιστρέψετε στην κανονική λειτουργία.

Άσκηση 5. Μελετήστε τη συνάρτηση f(x) = 1 - sqrt(x)/2

0. Ξεκινήστε την εφαρμογή ή πατήστε Control συν N για να ξεκινήσετε μια νέα συνάρτηση εάν έχετε ήδη μελετήσει μια έκφραση
1. Πατήστε 1 μείον s q r t ( x ) / 2
2. Πατήστε F3 και βεβαιωθείτε ότι έχετε επιλέξει το "χρήση μουσικών νότων"
3. Πατήστε F4 και βεβαιωθείτε ότι έχετε επιλέξει το "χρήση εναλλακτικών νότων για αρνητικές τιμές".
4. Πατήστε F5 και βεβαιωθείτε ότι έχετε επιλέξει το "Πάυση ήχου στο μηδέν".
5. Πατήστε Enter. Θα ακούσετε θόρυβο. Αυτό σημαίνει ότι η συνάρτηση δεν ορίζεται σε αυτό το σημείο. Κάποια στιγμή θα ακούσετε τον ήχο για τα σημεία που ορίζεται η συνάρτηση.
6. Πατήστε Home. Θα ακούσετε ότι βρισκόμαστε στο σημείο εκκίνησης.
7. Πατήστε Ctrl + x. Θα ακούσετε -10.00
8. Πατήστε Ctrl + y. Θα ακούσετε "δεν ορίζεται". Αυτό σημαίνει ότι η συνάρτηση δεν έχει οριστεί σε αυτό το σημείο
9. Πατήστε Control + δεξί βέλος για να μεταβείτε στο επόμενο σημείο μελέτης. Θα ακούσετε θόρυβο και μετά θα ακούσετε "μέγιστο μετά από απροσδιόριστο σημείο".
10. Πατήστε Control + x. Θα ακούσετε 0,00
11. Πατήστε Control + y. Θα ακούσετε 1. Άρα η συνάρτηση δεν ορίζεται στο (-10,0). Ξεκινά από το (0,1) όπου υπάρχει μέγιστο.
12. Πατήστε Control + δεξιά. Θα ακούσετε "μηδέν".
13. Πατήστε Control + x. Θα ακούσετε 4.00
14. Πατήστε Control + y. Θα ακούσετε 0,00. Άρα η συνάρτηση έχει ρίζα στο (4, 0).
15. Πατήστε Control + δεξιά. Θα ακούσετε εναλλακτικές μουσικές νότες που σημαίνει ότι πάμε κάτω από το μηδέν. Τέλος θα ακούσετε το "τελικό σημείο".
16. Πατήστε Control + x. Θα ακούσετε 10
17. Πατήστε Control + y. Θα ακούσετε -0,58
18. Άρα η συνάρτηση δεν ορίζεται στο (-10,0). Ξεκινά από (0, 1) και κατεβαίνει στο (10, -0,58)
19. Πατήστε Home. Είμαστε πάλι στην αρχή.
20. Πατήστε Shift + δεξί βέλος. Θα ακούσετε "μέγιστο μετά από απροσδιόριστο σημείο". Το Shift + δεξί βέλος σας μεταφέρει στο επόμενο σημείο μελέτης χωρίς να χρειάζεται να ακούσετε ήχο. Το Shift + αριστερά σας μεταφέρει στο προηγούμενο σημείο μελέτης. Επομένως, το shift συν βέλος είναι ένας πολύ γρήγορος τρόπος για να εξερευνήσετε τη συνάρτηση.
21. Πατήστε Control + x. Όπως και πριν, θα ακούσετε 0,00 γιατί είμαστε στο μέγιστο σημείο, όπου ξεκινά η συνάρτηση.
22. Πατήστε Control + y. Θα ακούσετε 1,00
23. Πατήστε Page Up. Έχετε μετακινηθεί λίγο προς τα δεξιά.
24. Πατήστε Page Down. Έχετε μετακινηθεί λίγο προς τα αριστερά. Χρησιμοποιώντας το Page Up και το Page Down μπορείτε να εξερευνήσετε τη συνάρτηση με μεγάλη λεπτομέρεια.
25. Σε οποιοδήποτε σημείο μπορείτε να ακούσετε πού βρίσκεστε χρησιμποποιώντας το Control + x και το Control + y. Μπορείτε να χρησιμοποιήσετε το κουμπί Home και το κουμπί End για να πάτε στην αρχή και στο τέλος της συνάρτησης.
26. Πατήστε Control + [. Θα ακούσετε ότι μειώσατε τo βήμα όταν εξερευνάτε. Το Control + [ κάνει την πλοήγηση πιο αργή αλλά πιο ακριβή.
27. Πατήστε Control + ]. Θα ακούσετε ότι αυξήσατε το βήμα όταν εξερευνάτε. Το Control + ] κάνει την πλοήγηση πιο γρήγορη αλλά λιγότερο ακριβή.

Άσκηση 6. Μελετήστε τη συνάρτηση f(x) = 1/x
